FES

Front End Suffolk

What?

Having Been a front end developer and being thoroughly inspired by both IPRUG and London Web Standards. I realized that amongst those who work in the web whatever their chosen language and server side system they all share a commion interface, the browser. Not only does this environment reflect the work of developers it also reflects the work of designers, copywriters, photographers and UI and UX designers.

To this end I’m organising a browser user group. A monthly meetup with organised talks on browser based topics, and more importantly a chance to learn and share ideas about how to create amazing experiences on the web.

Where?

This has yet to be arranged.

When?

A weeknight in the next month, this will probably be open to negotiation to make sure the maxiumum amount of people can make it.

Why?

The state of the browser is in constant flux, new fashions skills and techniques are always coming to light, why spend so much time learning them independently? We can teach each other and collectively become better at our chosen skill, be it a designer learning to master javascript to create dynamism in their designs, or a back end developer starting to apply Object Orientated thinking to CSS.

Who?

This project, like many others is intended to be open to contributions from anyone with something to say, I (Anders) am keep on helping to build a closer digital community in suffolk.

This project is aimed at those who work with the web so, designers, developers and enthusiasts!

How?

Just get in contact with us via email or twitter!
